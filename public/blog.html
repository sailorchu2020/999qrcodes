August 28, 2025 - 999qrcodes

Hey! This is the first official blog post. Very exciting! For this first blog topic, I'd like to secure the beginning of "999qrcodes". However, this "999qrcodes" isn't referring to this site. Let me go into depth…

In my health class recently, we were taught about goals. How to create them, set a plan, putting in the work to reach that goal, etc. While it was a very straightforward lesson, it made me realize that I've never truly set a goal for myself, let alone commit to one. I don't mean something like getting homework done by the end of the week, it's more in the sense of a personal project for self-fulfillment. So, I decided to come up with one.

I want to study computer science, and I thought it'd be fun to work on a long-term programming project before college application season really kicks in. One of my favorite video games of all time is 9 Hours, 9 Persons, 9 Doors. Combining the two, we now have my new proposal: "9 Weeks, 9 Languages, 9 Projects". Cool, right? …Riiiight?

This project will span 9 weeks, and it will start on the first day of September, the 9th month of the year. The goal is to learn the basics of 9 programming languages and create 9 projects in 9 weeks. I'll dedicate each week to learning about one programming language, and create a small project by the end of the week using it. Thus, we have 999qrcodes. 9 [Weeks], 9 [Languages], 9 [Projects] [that] QR codes. (My initials are QR.)

I expect to reach my goal by October 19. I also intend to include documentation of the project in the Archive when it's finally ready. I'm excited, but hope I can really commit to this. My main reason for making this the first post is that if I don't solidify it somewhere, it'll decrease the chance I'll stay interested in it. Since this is posted for many to see, I expect to feel a bit pressure to keep up with the project.

That's all. Thank you for reading!

September 7, 2025 - Week 1

I meant Javascript, not C++!


I've decided to modify the 999qrcodes project to be more… realistic. It took me a while to get to where I am with HTML and CSS (and even then I'd barely consider myself an amateur), so why should I assume that I could learn the basics of a new coding language every week? I just don't want to make it too hard on myself. However, I still want to pursue some sort of project to improve my skills.


So, here is my new proposal: 9 Weeks, 9 Projects, 9 Features. Instead of studying different languages every week, I figured it would be better if I continued to study the ones I'm already somewhat familiar with (HTML, CSS) and develop more of that frontend knowledge with Javascript, which I know very little about. It's definitely less stressful that way, and in the end, this site benefits as well.


With that being said, it's the last day of the first week, and the first project is done! I created this QR code generator. It was a simple and straightforward project that I think is a great starting point. I feel like it's going to take a while for me to grow comfortable with Javascript's syntax, but that's the whole point of this! I was not only super busy but also very ill this entire week, so I'm grateful that I was able to get this out against all odds. See you next week!
